# AgroHub - Direct Farmers Marketplace & Smart Grow AI

AgroHub is a modern, responsive web application designed to connect farmers directly with buyers, eliminating middlemen and helping farmers maximize their profits. It also includes an offline crop recommendation engine to help farmers decide what to grow based on environmental and soil conditions.

## 🚀 Features

- **Direct Farm-to-Table Marketplace:** A digital market where buyers can browse fresh produce, manage a shopping cart, and place orders directly with farmers.
- **Smart Grow AI (Crop Recommendation Engine):** An algorithm that analyzes soil type, temperature, rainfall, pH, and market trend scores to recommend the top 3 most suitable and profitable crops for cultivation.
- **Interactive Farmer Dashboard:** Dynamic graphs and statistics powered by Recharts, showing sales, pending orders, and crop distribution.
- **Persistent State:** Uses React Context API and client-side LocalStorage to persist user profiles, products, carts, and order histories.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Library:** React (with Context API for global state)
- **Languages:** TypeScript, HTML5, CSS3
- **Styling:** Tailwind CSS, Shadcn UI components
- **Data Visualization:** Recharts
- **Icons:** Lucide React
